9.Travel-Mini-Program开源丸子小程序

二、使用建议

WordPress需安装一款插件：丸子小程序开源免费版

后台插件中心：Mini Program API

小程序安装指南：https://www.imahui.com/minapp/2185.html

前端文件操作排除分类：

在 api.js 里找到 getCategories ，把 ‘/wp-json/wp/v2/categories?orderby=id&order=asc’ 改成 ‘/wp-json/wp/v2/categories?exclude=排除的分类ID&orderby=id&order=asc’ 。多个排除的分类，用英文逗号隔开。

前端文件说明：

weTravel 为微信小程序前端，qTravel 为 QQ 小程序前端

beTravel 为某度小程序前端，ttTravel 为某条小程序前端
